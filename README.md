# Student Performance Prediction

Ce projet analyse et prédit les performances académiques d’élèves à partir de facteurs **démographiques**, **comportementaux** et **socio-économiques**.  
L’objectif est d’identifier les variables les plus influentes sur le **`Exam_Score`** et de tester plusieurs modèles de machine learning.

---

## Dataset Overview

- **Nom :** Student Performance Factors  
- **Taille :** 6 607 élèves  
- **But :** Comprendre les déterminants de la réussite scolaire et explorer leur impact sur la performance finale.

---

## Variables du Dataset

### Features (extraits)

| Variable | Description |
|---------|-------------|
| **Hours_Studied** | Heures d’étude par semaine |
| **Attendance** | Taux de présence en classe (%) |
| **Parental_Involvement** | Implication parentale (Low/Med/High) |
| **Access_to_Resources** | Accès aux ressources éducatives |
| **Extracurricular_Activities** | Activités extrascolaires (Yes/No) |
| **Sleep_Hours** | Heures de sommeil |
| **Previous_Scores** | Scores académiques antérieurs |
| **Motivation_Level** | Motivation (Low/Med/High) |
| **Family_Income** | Niveau de revenu familial |
| **School_Type** | Type d'école (Public/Private) |
| **Learning_Disabilities** | Troubles d’apprentissage (Yes/No) |
| **Parental_Education_Level** | Niveau d’études des parents |
| **Gender** | Genre |

> Le dataset contient **19 variables explicatives** au total.

---

### Target Variable

| Variable | Description |
|---------|-------------|
| **Exam_Score** | Score final de l’élève (0–100) |
